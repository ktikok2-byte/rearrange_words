- 스마트폰, 컴퓨터 어디서든 온라인으로 접속해서 단어 순서를 맞춰서 문장을 완성하는 게임을 만들고 싶어. 
- 나중에 스마트폰 앱으로도 출시할 예정이기 때문에 상단에는 광고가 위치할 배너 공간도 미리 만들어놨으면 좋겠어. 디자인 레이아웃을 고려하기 위해서 말이야. 
- 데이터는 한글과 영어가 세트로 있는 것으로 시작하고 싶어. 
- 처음에는 학습을 원하는 언어의 단어가 무작위로 나오고, 
- 타이머는 단어의 개수 만큼 1초씩 쟀으면 좋겠어. 그러니까 단어가 3개면 3초안에 맞춰야하는거지. 
- 3초안에 못 맞추면 자동으로 틀렸다고 체크하고, 원래 문장과, 네이티브 언어, 한글 문장을 같이 보여주는거지.
- 로그인화면이 있었으면 좋겠어. 그래서 유저별 정보를 저장해서 유저가 자기 공부 상황을 체크할 수 있도록 말이야. 
- 문제를 풀면, 
 - 그 문제를 맞혔는지, 틀렸는지, 
 - 문제의 단어개수
 - 문제를 맞췄다면 맞추는데 걸린 시간
 - 문제를 풀기 시작한 날짜(시분초)
 - 기타 너가 생각하기에 있으면 좋을 것 같은 정보.
- 문제를 맞추면 맞춘 문장과, 사용자의 모국어로 써진 문장을 같이 보여주고 다음 문제로 넘어갈지 선택하는 버튼도 보여줘. 문제를 맞추는데 걸린 시간도 함께.
- 사용자의 정보에는
 - 전체 푼문제수
 - 전체 틀린 문제수
 - 전체 맞은 문제수
 - 일주일간 푼 문제수
 - 일주일간 맞춘 문제수
 - 10 주간 푼 문제수
 - 10 주간 맞춘 문제수
 - 10 주간 틀린 문제수
 - 1년간 푼 문제수
 - 1년간 맞춘 문제수
 - 1년간 틀린 문제수
 - 1일간 푼 문제수
 - 1일간 맞춘 문제수
 - 1일간 틀린 문제수
 - 현재 레벨
 - 기타 너가 생각하기에 있으면 좋을 것 같은 정보가 있게 해줘. 
- 처음에는 1~3개의 단어로 이루어진 문장을 풀게 하고, 2번 연속 맞출 때마다 레벨이 올라가도록 해줘.  
 - 2번 연속 틀리면 레벨이 내려가게 해줘. 
- 레벨이 올라갈 수록 한 문장에 들어가는 단어의 수가 3개씩 증가하는거야.
 - 1레벨 : 1~3
 - 2레벨 : 4~6
 ... 
 이런 식으로 말이야.
- 그리고 문제를 풀때 모드를 선택할 수 있도록 했으면 좋겠어.
  - 모든 문제 풀기
  - 틀린문제 풀기
  - 맞은 문제 풀기
  - 안풀었던 문제 풀기
레벨은 안풀었던 문제 풀기에서만 적용되도록 해줘.
- 통계를 볼수 있는 화면도 있으면 좋겠어. 
 - 전체, 1년, 10주, 1주, 하루 동안 내가 몇개를 풀었고, 몇개 맞았는지, 숫자와 그래프로 보여줬으면 좋겠어. 
 - 그래서 매일 매일 내 정답률이 어떻게 변화하는지 알 수 있었으면 좋겠어. 
 - 통계에는 안풀었던 문제 풀기를 했던 데이터만 보이게 해줘. 
- 틀린문제 풀기를 해서 맞추면 그 문제는 틀린문제 풀기에는 더이상 등장하기 않고 맞은 문제 풀기에만 등장하게 해줘. 틀린문제 풀기를 해서 맞췄다고 해도, 그건 통계에 반영되지 않게 해줘. 
- 사용자가 회원가입을 할때, 본인의 모국어와 공부하고 싶은 언어를 선택할 수 있도록 했으면 좋겠어. 
- 이 외에도 다른 있으면 좋은기능이 있다면 너가 추가해줘. 
- 데이터는 영어와 한글이 모두 있는 네이버 사전의 example을 활용하면 어떨까 싶은데 다른 좋은 소스가 있으면 활용해도돼. 필요한 데이터를 미리 다운 받아도 되고, api나 다른 방법으로 실시간으로 해도돼. 하지만 무료로 가능했으면 좋겠어. 
- 만약 네이버 사전의 데이터를 쓸 예정이라면 url 구조는 다음과 같아. 
 - https://en.dict.naver.com/#/search?range=example&query=a
 - https://en.dict.naver.com/#/search?query=b&range=example
 이런 식으로 a 나 b를 검색하면 예문 페이지의 url이 결정돼.
  - 그런데 이 중에서 두 언어가 모두 있는것만 데이터를 가져와야돼.
